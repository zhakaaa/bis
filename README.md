# Basic Information System

This repository contains a semester-long project developed for the **Basic Information Systems** course.

Unlike traditional coursework with separate assignments, this project was built progressively from scratch throughout the semester. As new topics and concepts were introduced during lectures, they were incorporated into the system, allowing the project to evolve step by step.

## Project Development

During the course, the project was continuously expanded with new features and system components, including:

* System analysis and requirements gathering.
* UML diagrams and system modeling.
* Database design and data management.
* API development and integration.
* Backend and frontend implementation.
* Documentation and project planning.

Each stage reflected the practical application of newly learned concepts, transforming the project from an initial idea into a more complete information system.

## Purpose

The purpose of this repository is to document the full development lifecycle of an information system, from planning and design to implementation and integration.

This project demonstrates my experience in applying software engineering principles, system analysis techniques, database design, API development, and collaborative project development in a structured academic environment.
